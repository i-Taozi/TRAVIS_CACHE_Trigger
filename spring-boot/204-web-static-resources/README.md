## web-static-resources
### 项目说明

web静态资源配置 + 访问实例工程 

- 通过配置指定自定义的静态资源路径
- 通过java代码，手动添加静态资源的映射关系
- 演示不同资源文件下，同名静态资源的优先级
- 演示第三方jar包内的静态资源访问case
