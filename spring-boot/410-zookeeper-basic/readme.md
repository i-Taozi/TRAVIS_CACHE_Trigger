zk 基本使用姿势

- 节点创建，存在判断
- 节点数据获取，修改
- 监听
- 节点删除

zk节点区分四种类型:

- 持久节点
- 持久顺序节点
- 临时节点（会话终端则回收）
- 临时顺序节点
